# Proyecto ESP32 con CycloneTCP

## Descripción del Proyecto

Este proyecto tiene como objetivo integrar y desarrollar aplicaciones utilizando la pila **CycloneTCP** sobre **ESP-IDF**. CycloneTCP es una pila de protocolos TCP/IP que permite la conectividad en dispositivos embebidos, facilitando la implementación de aplicaciones de red en el ESP32.

## Estado Actual

El proyecto se encuentra en un estado de **trabajo en progreso** con integración parcial de CycloneTCP. Se están realizando pruebas y ajustes para asegurar la funcionalidad y estabilidad de la aplicación.

## Prerrequisitos

- **ESP-IDF**: Se recomienda utilizar la versión 4.2 o superior.
- **Toolchain**: Asegúrate de tener configurado el entorno de desarrollo de ESP-IDF, incluyendo las herramientas necesarias para compilar y flashear el firmware en el ESP32.

## Cómo Compilar y Flashear

Para compilar y flashear el proyecto, utiliza los siguientes comandos:

```bash
idf.py build
idf.py flash
```

## Hardware

Este proyecto está diseñado para ser ejecutado en el **ESP32**. Asegúrate de tener el hardware adecuado para realizar las pruebas y la implementación de la aplicación.
